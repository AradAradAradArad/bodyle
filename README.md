# Body Guess - משחק אנטומיה

משחק אינטראקטיבי ללימוד אנטומיה, בהשראת המשחק Travle. המשחק מיועד לסטודנטים לרפואה.

## איך לשחק

1. בחר רמת קושי:
   - **עצמות (קל)** - חיבור בין עצמות בגוף
   - **איברים (קל)** - חיבור בין איברים פנימיים
   - **שרירים (בינוני)** - חיבור בין קבוצות שרירים
   - **כלי דם (קשה)** - חיבור בין כלי דם
   - **עצבים (קשה)** - חיבור בין עצבים

2. בחר נקודת התחלה מהתפריט הנפתח

3. המשחק יבחר אוטומטית נקודת סיום

4. נסה לנחש את כל המבנים האנטומיים במסלול הקצר ביותר מנקודת ההתחלה לנקודת הסיום

5. יש לך 7 ניחושים ו-3 רמזים

## תכונות

- 5 רמות קושי שונות
- אלגוריתם חיפוש מסלול קצר (BFS)
- מערכת רמזים
- ממשק משתמש בעברית עם עיצוב כהה
- ויזואליזציה של המסלול

## התקנה

פשוט פתח את הקובץ `index.html` בדפדפן. אין צורך בשרת או בהתקנות נוספות.

## קבצים

- `index.html` - קובץ HTML ראשי
- `styles.css` - עיצוב וסגנון
- `script.js` - לוגיקת המשחק
- `anatomy-data.js` - מבנה הנתונים של החיבורים האנטומיים

## פיתוח עתידי

- הוספת מבנים אנטומיים נוספים
- שיפור הוויזואליזציה של הגוף
- הוספת מודים נוספים
- מערכת ניקוד מתקדמת
